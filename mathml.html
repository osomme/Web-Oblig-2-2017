Mathematical Markup Language

MathML er et markup språk som representerer matematiske formeler på en datamaskin. Disse språkene gjør det lettere å kommunisere matematiske formler på tvers forskjellige programvarer. MathML er en del av HTML5 og blir også brukt av andre programmer som blant annet OpenOffce. En liste over programmer som bruker eller kan redigere matematiske uttrykk via MathML kan finnes her.

Ikke alle nettlesere støtter MathML. For å få støtte på mest mulige nettlesere, kan et javascript library som heter MathJax brukes. Dette er nødvendig hvis man skal vise MathML på Blink browsere, som blant annet Chrome og Opera.

Formlene nedenfor ble skrevet i programmet MathType, der man enkelt kan skrive inn en matematisk formel, før programmet eksporterer MathML koden som så kan skrives inner på nettsiden.
